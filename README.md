Scrape stats.nba.com for NBA data.

## Install Requirements

Python 3 is required to scrape data. Install the project's dependencies with:

```shell
pip3 install -r requirements.txt
```

## Scrape data

```shell
chmod +x scrape.sh && ./scrape.sh
```
